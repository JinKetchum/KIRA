# KIRA 🤖
### Kinetic Intelligence & Response Assistant

A fully-featured personal AI assistant built with Python.

## Features
- 🧠 AI Brain powered by Groq (LLaMA 3.3)
- 🎤 Voice + Text input
- 🖥️ PC Controls (open apps, shutdown, lock, screenshot)
- 📧 Email sending
- 🌤️ Weather reports
- 📰 News updates
- 📝 Note taking
- 📅 Calendar management
- 📂 File management
- 😴 Wake word (KIRA sync)
- 🎨 Sci-fi GUI with animated plasma orb
- 🧠 Persistent memory across sessions

## Setup
1. Clone the repo
2. Create virtual environment: `python -m venv kira_env`
3. Activate: `kira_env\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Create `.env` file with your API keys
6. Run: `python kira_gui.py`
